```
class Solution {
    public int removeDuplicates(int[] nums) {
        if(nums.length == 0) return 0;
        int length = 0;
        int temp = nums[0];
        for(int i = 0;i < nums.length;i++) {
            if(nums[i] != temp) {
                length += 1;
                temp = nums[i];
                nums[length] = temp;
            }            
        }
        return length+1;
    }
}
```
就是两个指针，length 和 i，length是实际要返回的长度，遇到不同的字母才加一
得益于array本身已经排好序了。
