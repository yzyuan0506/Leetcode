```
class Solution {
    public void reverseString(char[] s) {
        if(s == null || s.length == 0) {
            return;
        }
        
        int end = s.length / 2;

        for(int i = 0; i < end; i++) {
            char temp = s[i];
            s[i] = s[s.length - i -1];
            s[s.length - i -1] = temp;
        }
        return;
    }
}
```

很简单，就是将string调转，利用array的index就行

注意java中 length 和 length() 的区别：

- length can be used for int[], double[], String[] 

- length() can be used for String, StringBuilder, etc
