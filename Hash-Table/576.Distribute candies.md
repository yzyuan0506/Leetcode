```
class Solution {
    public int distributeCandies(int[] candies) {
        Map<Integer, Boolean>sisterMap = new HashMap<>();
        int count = 0;
        int len = candies.length;
        int max = len/2;
        for(int i = 0; i < len; i++) {
            if(count == max)
                break;
            if(!sisterMap.containsKey(candies[i])) {
                sisterMap.put(candies[i], true);
                count += 1;
            }
        }
        return count;
    }
}
```

比较简单，sister的最多的糖果种类肯定不能超于糖果的一半
