```
class Solution {
public:
    int climbStairs(int n) {
        if(n == 0) return 0;
        vector<int> dp (n+1, 0);
        dp[0] = 1;
        dp[1] = 1;
        for(int i = 2; i <= n; i++) {
            dp[i] = dp[i-1] + dp[i-2];
        }
        return dp[n];
    }
};
```

Why DP works?

two ways to climb stair n:
1. n step = 1 step + n-1 step
2. n step = 2 step + n-2 step
