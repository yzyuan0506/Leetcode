### Method 1: depth first search (post order)
```cpp
class Solution {
    TreeNode* prev = NULL;
public:
    void flatten(TreeNode* root) {
        if(!root)   return;
        flatten(root->right);
        flatten(root->left);
        root->right = prev;
        root->left = NULL;
        prev = root;
    }
};
```

看题目要求是in-place, 就要考虑用指针来解决，不必用stack，queue之类的数据结构

Time complexity: O(n)
Space complexity: O(1)

### Method 2: Inspired by Morris Traversal(imperative)
```
void flatten(TreeNode *root) {
    while(root){
        if(root->left == NULL)
            root = root->right;
        else {
            if(root->right){
                TreeNode *l = root->left;
                while(l->right) l = l->right;
                l->right = root->right;
            }
            root->right = root->left;
            root->left = NULL;
            root = root->right;
        }
    }
}
```
